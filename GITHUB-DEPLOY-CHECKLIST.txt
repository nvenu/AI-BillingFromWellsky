GitHub Deployment Checklist - billingfromwellsky.solifetec.com
================================================================

LOCAL MACHINE
-------------
[ ] Navigate to project: cd /path/to/AI-BillingFromWellsky
[ ] Initialize git: git init (if not done)
[ ] Add files: git add .
[ ] Commit: git commit -m "Initial commit"
[ ] Create GitHub repository (private recommended)
[ ] Add remote: git remote add origin https://github.com/YOUR_USERNAME/kinnser-billing-automation.git
[ ] Push: git push -u origin main
[ ] Verify .gitignore excludes: .env, node_modules/, dist/, downloads/, *.pdf, *.xlsx
[ ] Create GitHub Personal Access Token (if private repo)

SERVER SETUP
------------
[ ] SSH into server: ssh user@your-server-ip
[ ] Check Git installed: git --version
[ ] Create directory: sudo mkdir -p /var/www/kinnser-billing
[ ] Set ownership: sudo chown $USER:$USER /var/www/kinnser-billing
[ ] Clone repo: git clone https://github.com/YOUR_USERNAME/kinnser-billing-automation.git kinnser-billing
[ ] Navigate: cd /var/www/kinnser-billing

INSTALL & BUILD
---------------
[ ] Install dependencies: npm install
[ ] Install Playwright: npx playwright install chromium
[ ] Install Playwright deps: npx playwright install-deps chromium
[ ] Build application: npm run build
[ ] Create directories: mkdir -p logs downloads

CONFIGURATION
-------------
[ ] Create .env: nano .env
[ ] Add KINNSER_USER
[ ] Add KINNSER_PASS
[ ] Add SMTP_HOST=smtp.office365.com
[ ] Add SMTP_PORT=587
[ ] Add SMTP_USER
[ ] Add SMTP_PASS
[ ] Add PORT=8080
[ ] Add NODE_ENV=production
[ ] Save .env file
[ ] Secure .env: chmod 600 .env
[ ] Verify .env: cat .env (check all values)

NGINX SETUP
-----------
[ ] Create config: sudo nano /etc/nginx/sites-available/billingfromwellsky.solifetec.com
[ ] Paste Nginx configuration (see QUICK-START-GITHUB.md)
[ ] Save config
[ ] Enable site: sudo ln -s /etc/nginx/sites-available/billingfromwellsky.solifetec.com /etc/nginx/sites-enabled/
[ ] Test config: sudo nginx -t
[ ] Reload Nginx: sudo systemctl reload nginx

SSL CERTIFICATE
---------------
[ ] Run Certbot: sudo certbot --nginx -d billingfromwellsky.solifetec.com
[ ] Follow prompts
[ ] Choose redirect HTTP to HTTPS
[ ] Verify certificate: sudo certbot certificates

DNS CONFIGURATION
-----------------
[ ] Log into DNS provider
[ ] Add A record:
    Type: A
    Name: billingfromwellsky
    Value: [Same IP as helperfunction.solifetec.com]
    TTL: Auto or 3600
[ ] Save DNS record
[ ] Wait 5-30 minutes for propagation
[ ] Test DNS: nslookup billingfromwellsky.solifetec.com

START APPLICATION
-----------------
[ ] Navigate: cd /var/www/kinnser-billing
[ ] Start PM2: pm2 start ecosystem.config.js
[ ] Save PM2 config: pm2 save
[ ] Check status: pm2 status
[ ] View logs: pm2 logs kinnser-billing-automation
